# Arena Core - Excalibur

Excalibur.js版の `Arena Core` です。

## Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Controls

- Move: `WASD` or arrow keys
- Aim: mouse
- Shoot: left click or `Space`
- Restart: `R`

## Notes

Excaliburの `Engine`、`Scene`、入力管理、Canvas Graphics Contextを使っています。比較のため、衝突とゲーム状態はPhaser版と近い手動ロジックにしています。
