# Arena Core - Kontra

Kontra.js版の `Arena Core` です。

## Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Controls

- Move: `WASD` or arrow keys
- Aim: mouse
- Shoot: left click or `Space`
- Restart: `R`

## Notes

Kontraの `init`、`GameLoop`、keyboard/pointer helpers を使っています。描画、ゲーム状態、衝突判定はCanvas上で手動管理しています。
