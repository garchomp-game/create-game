# Phaserローカルログ

Vite開発サーバーは、開発診断用の詳細ランJSONをこのフォルダへ保存します。

- 手動ラン: `runs/*.json`、最新200件。
- デバッグラン: `debug/*.json`、最新100件。
- 明示的な自動テスト: `tests/*.json`、最新20件。
- JSONファイルはgit管理外です。
- `npm run dev`で起動し、1秒以上のランがゲームオーバーになると起点別に保存します。
- 通常の自動ブラウザテストと1秒未満のランは、自動保存しません。
- プレイヤー向け履歴とランキングは版付きブラウザ保存を使い、このフォルダを読みません。

容量確認:

```bash
du -sh logs
find logs -maxdepth 2 -type f -name '*.json' -printf '%p %s bytes\n'
```

保持上限を超えた古いファイルは、次の保存時に起点別で削除されます。
