# Arena Core Audio Assets

このフォルダの音源は、Node.jsの決定的な波形合成とFFmpegのVorbisエンコードで、本プロジェクト向けに生成したものです。第三者の楽曲、録音、サンプル素材は含みません。

2026-07-11に手動プレイ所感を受け、短い単音中心の初版から次の構成へ更新しました。

- BGMを8秒から32秒へ延長し、4区間でアルペジオ、旋律、打楽器密度を変化させた。
- 射撃、命中、撃破、回収、被弾へ複数の音色と微小なピッチ差を用意した。
- レベル上昇、強化、ゲームオーバーは、役割を判別しやすい固有フレーズにした。
- 初回遠征の勝利リザルトには、敗北SEと重ならない専用ループを用意した。

## 再生成

FFmpegが利用できる環境で、次を実行します。

```bash
npm run audio:generate
```

生成処理は`scripts/generate-audio.mjs`にあり、同じソースから同じ音色構成を再生成できます。

- 出典: 本リポジトリ内での手続き生成。
- 外部素材: なし。
- 第三者ライセンス表記: 不要。
- 再配布条件: リポジトリ全体のライセンス方針へ従う。現時点では音源単独へCC0などの追加ライセンスを設定しない。
