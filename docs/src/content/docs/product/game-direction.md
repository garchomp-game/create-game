## 基本方針

`Arena Core` は、1画面の見下ろし型アリーナサバイバルゲームです。

プレイヤーはアリーナ内を移動し、敵を倒し、XPを拾い、upgradeでrun中に成長しながら生存時間を伸ばします。

## 今後の方向性

- 実力で伸びる余地を残す。
- ただし、操作負荷だけで難しい状態にはしない。
- 1 runごとの判断を増やす。
- item、stage、equipment、meta progressionは段階的に足す。
- 課金なしのFree-to-Playとして、上達、選択肢解放、挑戦で継続性を作る。

## F2P方針

課金、ガチャ、広告、ログイン、バックエンドは当面扱いません。

やりこみは、次の要素で作ります。

- stage unlock
- equipment unlock
- challenge clear
- collection / profile stats
- run modifier
- upgrade synergy

恒久攻撃力や恒久HPだけで難易度を潰す設計は後回しにします。
