## 1回のラン

1. モード、ステージ、開始武器を選ぶ。
2. アリーナへ入る。
3. 移動と照準を行い、敵を倒す。
4. XPや回復ピックアップを回収する。
5. レベル上昇時に強化を選ぶ。
6. 敵構成、出現方向、危険度の変化へ対応する。
7. HPが0になるか、ステージ条件を達成するとランが終わる。
8. リザルトでスコア、死因、ビルド、自己記録との差を確認する。

現在はエンドレスモード相当の単一アリーナだけが動作し、ステージ選択と開始武器選択は未実装です。

## 判断の時間間隔

| 時間幅 | 主な判断 |
| --- | --- |
| 1から5秒 | どこへ避け、どの敵を狙うか |
| 10から30秒 | ピックアップを取りに行くか、防御行動を温存するか |
| 30から90秒 | ビルドをどの方向へ伸ばすか |
| 2から4分 | 特殊な敵構成やエリートへどう備えるか |
| 6から10分 | ボス撃破、ステージクリア、追加危険度のどれを狙うか |

敵数が増えるだけの時間を長くせず、長時間ランでも最後の意味ある判断から90秒以上空かない状態を目標とします。

## ランをまたぐループ

1. リザルトと履歴から改善点を確認する。
2. 別の武器、強化方針、ステージ、チャレンジを選ぶ。
3. 達成条件によって新しい選択肢を解放する。
4. 自己記録、ステージ記録、武器熟練度を更新する。

恒久攻撃力を積み上げることより、新しい遊び方と挑戦を増やすことを優先します。

## 現在の拡張点

- ステージごとの地形、敵構成、クリア条件。
- 開始武器と武器固有強化。
- 短時間アイテムと回復判断。
- エリート、ボス、特殊な敵出現パターン。
- 危険度と報酬を同時に上げるラン契約。
- チャレンジ、武器熟練度、プロフィール統計。
