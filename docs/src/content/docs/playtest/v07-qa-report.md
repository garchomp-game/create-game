## 手動所感からのRC2変更

RC1のPulse 2ランでは、序盤敗北1件、約38,000点勝利1件でした。不可避攻撃はありませんでしたが、射撃イベントとボス直前が最難関で、ボスが道中より易しい難度逆転がありました。

RC2では次を変更しました。

- 敵種を0秒、75秒、180秒、300秒で段階解禁する専用ウェーブ。
- XP 1.75倍、スコア1.5倍、回復抽選1.35倍により、少ない同時敵数でもビルド進行を維持。
- ボス開始時に道中の敵と敵弾を退避。
- 護衛ターンにも固定照準の抑制斉射を追加し、第2段階で弾数と護衛数を増加。
- 勝利専用12.8秒BGMと、勝利時に敗北SEを鳴らさない遷移。

## 可避性と説明可能性

- Pulse / Spreadの両方で固定照準斉射を無被弾回避するfixtureが通る。
- 挟撃増援は安全距離、活動領域、障害物、到達可能性、敵上限を共有する。
- 挟撃直後の不可避接触がなく、随伴接触は`escort-pincer`へ帰属する。
- 斉射弾は`targeted-salvo`へ帰属し、最後の攻撃種をリザルトとRun JSONへ残す。
- 護衛ターンの抑制斉射は`escort-pincer`へ帰属する。
- ボス撃破による勝利、保存、`game.over`は一度だけ発生する。
- ボスfixtureは静的背景1回、dynamic 8ms未満、screen / HUD 5ms未満、feedback 3ms未満の描画契約を通る。

## 既知事項

- 自動6ランではSpreadの完了がPulseより約31秒から66秒遅い。全条件が10分30秒以内で、スコア帯は重なるためpreview停止条件にはしない。
- 展開空白は全seedで目標90秒以内になった。
- ボス専用背景、最終アート、会話、恒久報酬はv0.7対象外である。
- 外部アカウント、オンラインランキング、自動解析、詳細ログ送信は導入していない。

## 手動採否

RC1の2本は調整根拠として残しますが、RC2のproduction採否ランには混ぜません。更新previewで各武器3本を、デバッグhookと観戦AIを使わずに行います。

記録する項目:

- ボス到達、勝敗、クリアまたは敗北時刻、スコア、到達Act。
- 最も崩れやすかったカード、敵、ボス攻撃。
- 予告を見て回避できたか。不可視、即死、操作不能がなかったか。
- Pulseの優先標的・射線維持とSpreadの群れ処理が別の強みとして感じられたか。
- 6分30秒までの展開とボス戦に、間延びまたは急な難度跳躍がなかったか。
- 再挑戦したい理由が、カード順、武器、操作改善のいずれかにあったか。

重大 / 高の再現不具合が0件で、両武器がボスへ到達し、少なくとも一方で勝利できればproduction昇格候補とします。数値差だけで即調整せず、6ランの所感とラン記録を合わせて判断します。
