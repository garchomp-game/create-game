## この設計の目的

開始武器を選んだ時点で、狙う対象、立つ場所、強化後の戦い方が変わる状態を作ります。

武器差を発射間隔や弾数だけに置くと、強化後に同じ射撃形状へ収束し、最終的には秒間火力の高い方が上位互換になります。共有強化は基礎能力を支えてもよい一方、別武器の初期形態を複製してはいけません。

v0.6.8では次を設計原則として採用します。

1. 武器固有強化は、その武器の初期操作を深める。
2. Pulseは単線の継続照準と反射角、Spreadは広角の複数標的を最後まで維持する。
3. 雑魚と高耐久標的を分け、両武器に異なる得意対象を作る。
4. 最終強化は取得前後で狙い方または地形利用を変える。
5. 強さは総合勝率だけでなく、役割別の成果で比較する。

2026-07-14の手動再評価では、Pulseで7972点 / 163秒と26355点 / 354秒を記録しました。後者は274.79秒で通常ビルドを完成しEX Lv3へ到達したため、分裂を戻さず、操作精度で成果上限が伸びる現在の武器契約を採用します。

## v0.6.2までの問題

変更前の分裂射撃は、すべての開始武器へ弾を1発ずつ加えていました。

| 武器 | 初期弾数 | 最大弾数 | 基礎射角 | 基礎間隔 | 最大時の基礎弾数/秒 |
| --- | ---: | ---: | ---: | ---: | ---: |
| Pulse | 1 | 3 | 0.12 rad | 0.16秒 | 18.75 |
| Spread | 3 | 5 | 0.52 rad | 0.32秒 | 15.63 |

同じ2ランクでも、Pulseは弾数が3倍、Spreadは約1.67倍です。成長後のPulseはSpreadの初期形態と同じ3方向射撃になり、狭角であること以外の形状差が薄れます。Pulseは弾速、射程、反響回路も持つため、Spreadの価値が広い射角だけに偏ります。

## 武器の役割契約

| 軸 | Pulse | Spread |
| --- | --- | --- |
| 射撃形態 | 単線の連続射撃 | 広角の斉射 |
| 主な判断 | 優先標的へ照準を維持する | 複数標的が射角へ入る位置を取る |
| 得意対象 | Brute、Ranged、縦に並ぶ敵 | Chaser、Fast、横に広がる群れ |
| 地形利用 | 射線、貫通、障害物・外周反射 | 開けた空間、包囲への扇状制圧 |
| 成長方向 | 集中ダメージと直線処理を深める | 弾幕幅と複数命中の循環を深める |
| 明確な弱点 | 横から広がる群れ | 高耐久単体への集中火力 |

どちらも通常ランを完走できる必要はありますが、同じ敵構成に対して同じ操作を要求しません。

Spreadは広角と弾数によって最低限の処理量を安定させる武器です。Pulseは同じ安定性を持たせず、高速弾を優先標的へ通し、最大4体の貫通順と反射後の射線を作れたときに上限が伸びる武器とします。難しいPulseを一律にSpreadより強くするのではなく、操作精度を成果へ変換できる余地を持たせます。

## v0.6.3手動ランからの再調整

同じ`phaser-v0.6.3-weapon-identities`で、Pulse 3本とSpread 1本を記録しました。

| 武器 | 生存時間 | スコア | 通常ビルド完成 | 観測 |
| --- | ---: | ---: | ---: | --- |
| Pulse | 121秒 | 4689 | 未到達 | 単線化後の処理量が低く、1万点へ届きにくい |
| Pulse | 171秒 | 8250 | 未到達 | 弾速強化は最大だが、完成前に終了 |
| Pulse | 146秒 | 6908 | 未到達 | 分裂を失った影響が大きい |
| Spread | 421秒 | 34434 | 277秒 | EX Lv5 / C2まで到達し、広角処理が明確に安定 |

この結果だけでPulseへ分裂を戻すと、再びSpreadの初期形態を複製します。v0.6.4では連射、敵HP・速度・出現、XP、集束共鳴を動かさず、Pulseだけが加速弾から得る弾速と、反響回路で利用できる反射面を増やします。これにより武器形状を維持したまま、照準精度、貫通順、反射角を成果へ変換します。15分耐久でのみ、射撃体集中時の可読性を守る同時敵弾256の安全上限を追加しています。

## v0.6.6の精密射撃調整

632.81秒 / 77883点のPulse手動ランはアリーナ崩壊1段階へ到達しましたが、同条件のSpreadと観戦AIの双方でPulseよりスコアを伸ばしやすい傾向が残りました。Spreadの群れ処理と爽快感は意図した役割なので弱体化せず、Pulseだけを次の範囲で補います。

- 基礎ダメージを1.00から1.16へ上げる。基礎射撃間隔0.12秒、弾数、弾速、貫通上限は変えない。
- 集束共鳴を1段階+20%とし、同じ敵への連続直撃に加えて、同じ弾が跳弾前に貫通した敵数でも加算する。
- 単体への最大4段階は+80%、最大集束時の単発は2.088とする。
- 貫通列の追加分は、その弾が先に直撃した敵数を使う。跳弾後の命中には集束と貫通列の追加ダメージを与えない。

未強化時の連射DPSはPulse約9.67、3発Spread約9.09です。通常強化後の単体理論DPSはPulse約17.4、5発Spread約15.15となり、Pulseは正確に当て続けたときだけ約15%上回ります。実戦ではSpreadが複数方向を同時に処理できるため、総合スコアを保証する差ではありません。

## v0.6.8の外周反射採用

比較観察で再有効化していたPulse外周反射を、公開ルールとして採用します。外周を外すとPulseの反響回路は4障害物だけに依存し、Spreadの掃射循環に比べて立ち位置の選択肢が薄くなります。一方、外周はPulse弾だけに作用し、Spread、敵、敵弾、プレイヤー、出現、崩壊領域を変えません。

直近の手動4ランでは、Pulseが33069点 / 413.10秒と25493点 / 349.79秒、Spreadが25382点 / 354.78秒と33093点 / 416.72秒でした。Pulseは合計7957回の外周反射から1332追撃命中・625撃破を得ています。両武器の到達帯を一方的に分けず、Pulseだけに地形判断の成果が出たため、数値変更なしで採用します。

この採否は手動4ランの平均勝率を保証する判断ではありません。公開ベータでは武器別分布を継続観測し、単一高得点を再調整根拠にしません。

## 強化カタログの分離

### 共通強化

- 連射強化: 射撃間隔を短縮する。Pulseは連続火力、Spreadは斉射間隔として体感が異なる。
- 軽快な足取り: 位置取りの余地を増やす。
- 生命コア: 生存余地を増やす。
- 加速弾: Spreadと開発用Pierceは1ランク15%、Pulseは1ランク22%弾速を上げる。5ランク時はPulse約1405px/s、Spread約965px/sになる。
- 貫通弾: Pulseでは直線処理、Spreadでは扇状の群れ処理を強める。

### Pulse固有強化: 集束共鳴

分裂射撃の代わりに、同じ敵へ直撃を重ねる集中強化を2ランク用意します。

単線化による序盤の処理量低下は、Pulseの基礎射撃間隔を0.16秒から0.12秒へ短縮して補います。最大強化後も弾数ではなく連射密度が伸びるため、Spreadの射撃形状を複製しません。

| 項目 | 仕様 |
| --- | --- |
| 対象 | 跳弾前のPulse直撃 |
| 継続時間 | 最後の直撃から0.9秒 |
| 1スタック | 次回以降の直撃ダメージ +20% |
| ランク1 | 最大2スタック、最大+40% |
| ランク2 | 最大4スタック、最大+80% |
| 解除 | 時間切れ、または対象の消滅 |

スタックは敵ごとに保持します。貫通弾が別の敵へ当たっても、それまでの対象の蓄積を消しません。同じ弾の2体目以降には、先に貫通した敵1体につき同じ+20%を追加し、直線へ敵を並べる操作へ報酬を与えます。跳弾後の命中は集束共鳴を蓄積せず、貫通列の追加分も与えません。反響回路は範囲補助として役割を分けます。

表示では対象の周囲へ最大4段階の短い共鳴目印を出し、色だけに依存せず蓄積を読めるようにします。

### Pulse最終強化: 反響回路

- 武器カテゴリ7ランクで解放する。
- 障害物または固定アリーナ外周で、合計1回跳弾する。
- 外周はPulse弾だけに作用する一方向反射フィールドとする。プレイヤー、敵、敵弾、画面外32pxの敵出現は従来どおり通過する。
- 600秒以降に縮小する安全領域の境界は反射面にしない。
- 跳弾前は集中処理、跳弾後は地形を使った追加処理として扱う。
- 取得時刻、障害物反射、外周反射、外周面別反射、跳弾後命中、異なる追加命中敵数を記録する。

弾速は射撃時に確定し、飛行中には加速しません。高速弾は1フレームの始点と終点だけで判定せず、移動線分上で敵と最初に交差する時刻を求めます。貫通は交差時刻順に処理し、障害物または外周へ到達した場合は残りフレーム時間を反射後の線分へ使います。

### Spread固有強化: 分裂射撃

- Spreadだけへ提示する。
- 基礎射撃間隔は0.33秒とする。
- 3発から4発、5発へ増やす。
- 0.52 radの広角を維持し、Pulseの単線射撃へ持ち込まない。
- 1斉射あたりの異なる命中敵数を主要成果とする。

### Spread最終強化: 掃射循環

1斉射で複数標的を捉えると、次の斉射が早まる最終強化です。

| 項目 | 仕様 |
| --- | --- |
| 解放 | 武器カテゴリ7ランク |
| 条件 | 同じ斉射が異なる敵3体へ命中 |
| 報酬 | 掃射チャージを1つ得る |
| 消費 | 次のSpread射撃時 |
| 効果 | 成立時点から次弾までの残り待ち時間を70%にする |
| 上限 | チャージは1つ。1斉射につき1回だけ発動 |

自動射撃中でも次の1回へ確実に適用し、命中時点で射撃待ち時間が残っていなくても報酬を失いません。HUDと取得フィードバックで準備・消費を示します。

## 通常ビルド完成条件

PulseとSpreadは、関連する通常強化の総ランクを同じにします。

| 区分 | Pulse | Spread |
| --- | ---: | ---: |
| 共通5強化 | 22ランク | 22ランク |
| 武器固有強化 | 集束共鳴 2 | 分裂射撃 2 |
| 最終強化 | 反響回路 1 | 掃射循環 1 |
| 合計 | 25 | 25 |

「完成ラン」はゲームクリアではなく、関連する通常25ランクを取得してEX進行へ入ったランを指します。文書とUIでは「通常ビルド完成到達ラン」と表記します。

## 敵耐久との対応

序盤の撃破感とXP速度を変えず、240秒以降に敵役割の差を広げます。基礎HPは変更しません。

| 敵 | 基礎HP | 45秒ごとのHP倍率 | 役割 |
| --- | ---: | ---: | --- |
| Chaser | 1 | 1.12 | 標準雑魚 |
| Fast | 1 | 1.10 | 柔らかい高速脅威 |
| Ranged | 2 | 1.15 | 優先して処理する中耐久標的 |
| Brute | 3 | 1.17 | 照準を維持する高耐久標的 |

600秒付近では現行値と比べ、ChaserとFastはおおむね同等、Rangedは約5から7、Bruteは約8から11HPになります。全敵を一律に硬くせず、Pulseの集中処理とSpreadの群れ処理が同じランに共存する構成を作ります。

HP増加で撃破数とXPが落ちるため、通常ビルド完成時刻、画面内敵数、接触被害を同時に監視します。難易度を上げることと、敵を倒す手応えを増やすことを混同しません。

## 高解像度の選択UI

PhaserのTextやSVGアセットはCanvasへ描画された時点でラスタライズされます。960 x 540のCanvasを拡大するだけでは、元がSVGでも動的な日本語文字はぼやけます。

武器、通常強化、EX強化、後半契約の選択画面は、Canvas上へ重ねるDOM/CSSレイヤーへ移します。

- ブラウザのフォント描画を使い、表示倍率とDPIごとに再描画する。
- 日本語フォントは`Noto Sans CJK JP`、`Noto Sans JP`、`Yu Gothic`、`Meiryo`、sans-serifの順で指定する。
- ボタンは実際の`button`要素とし、ポインター、Tab、Enter、数字キーへ対応する。
- デスクトップは最大3列、縦長画面は縦並びにする。
- Canvasの論理座標へ依存せず、狭い画面でも文字を読める大きさに保つ。
- 画面比較はCanvas単体ではなく、ゲーム親要素を撮影する。

ゲーム中のHUDと弾・敵表示はCanvasへ残します。DOM化は選択とメニューの可読性を優先し、シミュレーション規則から分離します。

## 計測契約

Pulse:

- 集束共鳴の強化命中数、追加ダメージ、最大スタック。
- 同じ敵への連続命中と、同一弾の貫通列について、強化命中数と追加ダメージを別々に保存する。
- 集束共鳴中の敵種別撃破。
- 反響回路の障害物反射、外周反射、外周面別反射、跳弾後命中、異なる追加命中敵数。
- 跳弾後命中・撃破を障害物と外周へ帰属し、外周追撃命中は左・右・上・下にも分ける。

Spread:

- 斉射あたりの異なる命中敵数。
- 掃射循環の成立回数、消費回数、成立までの平均時間。
- 敵種別命中と撃破。

共通:

- 毎分撃破、毎分スコア、生存時間、被ダメージ。
- 通常ビルド完成時刻、EX開始時刻。
- 敵1体あたりの平均命中数と敵種別の命中数。
- 画面内敵数の最大値と上限滞在時間。

## 採否基準

- Spreadの命中斉射あたり異なる敵数がPulseより15%以上高い。
- PulseのBrute / Rangedへの追加ダメージが記録され、集束共鳴ランク2で最大スタックへ到達できる。
- 自動10シード比較で、生存時間、毎分撃破、毎分スコアの武器差がそれぞれ20%以内に収まる。
- 最大弾速はPulse約1405px/s、Spread約965px/sで、50msステップでもFastをすり抜けない。
- 貫通は移動線分上の手前から順に処理し、同一フレームで往路命中、外周反射、復路命中が成立する。
- 240秒以前の無入力死亡時間と最初の強化時刻を現行許容範囲へ保つ。
- 通常ビルド完成は4分から6分を維持する。
- Pulse手動再評価で1本以上が240秒から360秒で通常ビルド完成へ到達する。26355点 / 353.96秒のランが274.79秒で満たした。
- 通常ビルド完成後の平均命中数は、直近Pulseランの1.89命中/撃破から2.1から2.6を最初の目標とする。
- 960 x 540、844 x 390、390 x 844、DPR 1 / 2で文字切れと選択不能がない。

数値を満たしても、両武器で同じ場所に立ち同じ方向へ撃つことが最適なら採用しません。手動プレイでは、Pulseの優先標的維持とSpreadの群れを射角へ収める移動が実際に生じたかを記録します。

## 実装境界

- アプリ版はv0.6.8、ルール版は`phaser-v0.6.8-pulse-boundary-ricochet`とする。
- 旧ラン記録は削除せず、新しい計測値を既定値付きで追加する。
- ラン途中保存はないため、旧強化ランクを新しいランへ移行しない。
- 開発用Pierceは今回の製品選択へ追加しない。
- 外部フォント配信へ依存せず、OSのフォールバックを使う。
- 武器固有効果、Pulse外周反射、DOM選択UI、敵耐久を個別の機能境界とテストで無効化できるようにする。
