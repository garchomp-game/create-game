## 目的

Practice Sandboxは、基本訓練を終えた後に、時間制限や難度上昇を気にせず
移動、照準、武器、敵役割を試すための自由練習モードです。
Endlessの攻略や競技記録を簡単にする支援機能ではありません。

## 現行候補

タイトルの「練習場」を開くと、大きなパルス / 拡散ボタンだけを優先して表示します。
条件項目はPractice開始後の画面上部にある「設定」へ分離します。
設定中はシミュレーションを停止し、閉じると同じ練習へ戻ります。
無敵は即時、出現量と敵種は以後のスポーンへ反映します。
設定名と変更値を分離し、段階値は左右矢印、敵種はチェックボックスで操作します。

| 設定 | 選択肢 | 既定値 |
| --- | --- | --- |
| HP | 無敵オン / オフ | オン |
| 敵の出現量 | 少なめ / 標準 / 多め | 少なめ |
| 敵の種類 | 追跡体 / 重装体 / 高速体 / 射撃体 | 追跡体と重装体 |
| 武器 | パルス / 拡散 | 開始時に選択 |

敵は最低1種類を必要とします。選択は同じブラウザセッション中の再開で保持し、
プロフィール設定やLocalStorageには保存しません。

## ランタイム契約

- `practice/practice-arena`をTrainingやEndlessとは別のmode / stageとして扱う。
- 1本の固定waveだけを使い、時間経過で出現数、速度、HPを上げない。
- 危険イベント、後半契約、アリーナ崩壊、EX Protocolを無効にする。
- `recordPolicy: none`とし、履歴、PB、ランキング、報酬を作らない。
- 無敵オンでは接触、敵弾などのプレイヤーダメージを適用しない。
- 無敵オフで終了した場合も、通常記録の保存失敗とは表示せず練習終了として扱う。

## 表示

戦場背景へ、`WASD`移動、マウス照準、射撃、一時停止の操作を低い強度で常時表示します。
敵や弾の可読性を優先し、説明パネルや進行チェックリストは重ねません。

## 今後の候補

ボス単体、敵の同時出現上限、敵速度、Pickup有無などは追加可能です。
ただし最初の候補では設定量を増やしすぎず、自由練習として必要な
「死なずに観察する」「苦手な敵だけ試す」を優先します。
