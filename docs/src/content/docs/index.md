`Arena Core` は、ブラウザで動く見下ろし型アリーナサバイバルゲームです。

このドキュメントは、比較実験として始まったプロジェクトを、Phaser版の本格的なゲーム開発ワークスペースへ整理するための入口です。

## 今の位置づけ

- 主対象は `phaser` 実装です。
- `v0.1` は公開済みbaselineです。
- ローカル `main` には v0.2 / v0.3 相当の改善が入っています。
- v0.3 candidateでは、回復pickup、画面外敵方向indicator、日本語UI、splitShotの小さな拡散が入っています。
- 次の開発焦点は v0.4 の操作性、障害物、弾と壁の関係です。
