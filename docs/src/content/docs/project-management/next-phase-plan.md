## 現在の判断

v0.6.4は、エンドレス後半の基盤へPulse高弾速、連続衝突判定、固定外周反射を加え、単体試験と武器別10シード比較まで完了しました。

現在の残作業は自動QAの最終確認と手動採否判定です。v0.6.3の手動比較ではPulse 3本が通常ビルド完成前に終了し、Spreadは34434点・約421秒でEXへ到達しました。v0.6.4ではPulseへ分裂を戻さず、最大約1405px/sの弾速、貫通順、障害物・外周反射で技能上限を広げています。次の3ランでは数値を動かさず、完成時刻、反射利用、死因を揃えて判断します。

- v0.6の個別要件は[v0.6チケット詳細](../v06-tickets/)を正本とする。
- v0.6以降のつながりは[中長期作業計画](../gameplay-expansion-plan/)を参照する。
- ゲームプレイ原則は[拡張設計の全体像](../../design/gameplay-expansion-blueprint/)を参照する。

## v0.6で解く問題

### 1. ビルドが同じ形へ収束して見える

v0.5の手動5ランはすべて開始武器が`pulse`でした。全ランで`splitShot`が最大ランク2、`vitalCore`はランク0で、直近の約301秒と322秒の2ランは最終強化構成が同一です。

v0.6.3ではPulseから分裂射撃を外して集束共鳴へ置き換え、Spreadだけを3発から5発へ成長させました。提示履歴と固有成果も保存するため、今後は候補運と選択意図を分けて比較できます。

### 2. 後半ほど意味のある選択が減る

v0.5ではXP必要量が`growth: 1.4`で増え、全強化後も空のレベルアップが発生していました。

v0.6では`growth: 1.25`、上限140へ変更してビルド完成を明示しました。v0.6.1では完成後もXP回収を続け、必要XP180から始まるEXレベルと反復強化へ接続しました。v0.6.3では25ランク化後も4〜6分で完成へ入るよう、通常XP上限だけを120へ調整しました。

### 3. 武器差をスコア以外で説明できない

武器別計測へ命中ボレー率、1射で命中した異なる敵数、敵種別命中・撃破、移動、反響回路成果を追加しました。自動入力では役割差が出ていますが、人間の操作差は未確認です。

### 4. 90秒以降の新しい判断が少ない

最初の射撃体集中を、射撃体集中、高速群襲来、重装包囲の反復デッキへ拡張しました。240秒の後半契約に加え、240秒以降の脅威ティアと600秒以降の崩壊を追加しています。各予告を識別して対応が変わるかを確認します。

## v0.6.1追加対応

1. 通常ビルド完成後のEXレベルと4種の限界強化。
2. 敵HP・攻撃力が上限なく伸びる脅威ティアと、画面密度側の安全上限。
3. 3種を一巡するまで重複しない危険イベントデッキ。
4. 600秒からのアリーナ崩壊と崩壊ダメージ。
5. EX、脅威、崩壊、イベント内訳の履歴・JSON・TSV / CSV計測。
6. `phaser-v0.6.1-endless-escalation`へのルール版分離。

設計値と採否条件は[エンドレス後半の継続圧力](../../design/endless-escalation/)を参照してください。

## v0.6.2追加対応

1. 射線が切れた敵だけが使う、決定論的な共有Dijkstra経路場。
2. 敵半径を含む障害物余白、目標セルの再利用、従来移動へのフォールバック。
3. 射撃体の遮蔽物越し発射を止め、射線を取るまで再配置する挙動。
4. 同一サイクル内で取得済み候補を除外し、最後の1件を自動取得するEXデッキ。
5. 過給サイクル+10%・超過駆動+6%を各5段階へまとめた強化幅。
6. `phaser-v0.6.2-navigation-extra-cycle`へのルール版分離と、経路・EXサイクル計測。

敵経路の詳細は[障害物・敵経路・弾](../../design/obstacles-and-projectiles/)を参照してください。

## v0.6.3追加対応

1. Pulse専用の集束共鳴と、Spread専用の分裂射撃。
2. Spread最終強化「掃射循環」と、Pulse / Spread各25ランクの対称ビルド。
3. 240秒以降のFast、Chaser、Ranged、Brute別HP成長。
4. 選択画面のDOM化、実ボタン、縦画面1列表示、DPR 2試験。
5. 集束、掃射、命中 / 撃破比のランJSON・CSV / TSV計測。
6. `phaser-v0.6.3-weapon-identities`へのルール版分離。

詳細は[武器アイデンティティ](../../design/weapon-identities/)を参照してください。

## v0.6.4追加対応

1. Pulseだけ加速弾を1ランク22%とし、最大約1405px/sへ伸ばす武器別効果合成。
2. 弾移動線分と敵円の連続衝突、貫通の交差時刻順処理。
3. 障害物または固定外周で合計1回反射し、残りフレーム時間を復路へ使う反響回路。
4. Pulse弾だけに作用し、敵出現と崩壊領域を変えない一方向外周フィールド。
5. 障害物、外周、外周面別反射の履歴・JSON・CSV / TSV計測。
6. `phaser-v0.6.4-pulse-ballistics`へのルール版分離。

### 5. 戦闘ルールに未確定箇所がある

- `hitCapacity`は1発の総命中可能数として表示と実装を統一した。
- 偶数の分裂弾は照準中央を空けて左右へ飛ぶ。
- 無敵時間中に重なった敵弾は、ダメージを与えず消滅する。

偶数分裂と敵弾消費は既存挙動を維持し、固定配置試験でルールを固定しました。

## 比較指標

スコアだけで武器や強化を評価しません。

- 強化の提示順、選択順、時刻、選択間隔。
- 1射あたりの異なる命中敵数と敵種。
- 毎分スコア、毎分撃破、被ダメージ源。
- 移動距離と危険イベント前後の移動方向。
- 最終強化の取得時刻、発動回数、追加命中。
- 開始武器別の生存時間分布。
- 手動所感として、照準距離、位置取り、標的優先が変わったか。

20%以上の一貫した差は強い採用信号としますが、数値差だけで面白さを断定しません。

## v0.6の終了条件

- PulseとSpreadの違いを、操作と強化方針の一文でそれぞれ説明できる。
- 長時間ランでも、成長が理由なく止まったように見えない。
- EX候補に無効果の選択がなく、敵成長を恒久的には追い越さない。
- 同じEX候補がサイクル内で重複せず、最後の自動取得がプレイを不必要に止めない。
- 敵が障害物へ張り付かず迂回し、射撃体が遮蔽物越しに発射しない。
- 挙動変化型の最終強化が、狙い方または地形利用を変える。
- 最大弾速でも敵をすり抜けず、貫通と反射角がPulseの技能上限として利用できる。
- 同じシードと開始武器で主要イベント列を再現できる。
- リザルトと履歴で、開始武器、強化順、最終強化を比較できる。
- 予兆付き危険イベントに対する反応を計測・説明できる。
- 3種の危険イベントを識別でき、同一シードで順序を再現できる。
- 熟練プレイでも有限時間で終了し、敵数と弾数は画面上限内に収まる。
- 実験ルール、デバッグ、自動テストが通常ランキングへ混ざらない。
- 自動比較、手動ラン、E2E、画面比較から採用・調整・削除を判断できる。

ステージ、エリート、ボスはv0.7で、戦闘展開制御と出現安全規則を作ってから扱います。
