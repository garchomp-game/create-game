# Arena Core Docs

Starlight documentation site for the Phaser version of `Arena Core`.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Content

- `src/content/docs`: Starlight pages
- `src/content/docs/project-management`: roadmap, tickets, decisions, risks
- `src/content/docs/archive/legacy`: migrated legacy Markdown notes

The playable game implementation lives in `../phaser`.
