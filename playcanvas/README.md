# Arena Core - PlayCanvas

PlayCanvas版の `Arena Core` です。

## Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Controls

- Move: `WASD` or arrow keys
- Aim: mouse
- Shoot: left click or `Space`
- Restart: `R`

## Notes

PlayCanvasをエディタなしのコードファーストで使い、Application、Entity、Camera、Light、Render componentsを直接構成しています。
