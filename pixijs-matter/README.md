# Arena Core - PixiJS + Matter.js

PixiJS + Matter.js版の `Arena Core` です。

## Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Controls

- Move: `WASD` or arrow keys
- Aim: mouse
- Shoot: left click or `Space`
- Restart: `R`

## Notes

PixiJSを描画、Matter.jsを物理Bodyと障害物衝突に使います。HUDはDOMで重ねています。
