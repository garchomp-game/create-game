# Arena Core - melonJS

melonJS版の `Arena Core` です。

## Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Controls

- Move: `WASD` or arrow keys
- Aim: mouse
- Shoot: left click or `Space`
- Restart: `R`

## Notes

melonJSの `Stage` と `Renderable` を使って、1つのゲーム画面内で共通仕様の更新・描画を行います。
