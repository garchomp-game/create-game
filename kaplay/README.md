# Arena Core - KAPLAY

KAPLAY版の `Arena Core` です。

## Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Controls

- Move: `WASD` or arrow keys
- Aim: mouse
- Shoot: left click or `Space`
- Restart: `R`

## Notes

KAPLAYの `add`、`circle`、`rect`、`area`、`text`、`onUpdate` を使い、ゲーム状態や衝突処理は共通仕様に合わせて手動で管理しています。
